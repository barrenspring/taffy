DM 看板截图放置说明 / How to add the DM dashboard screenshots
================================================================

述职报告的「管理创新」这一页内置了一个 DM 看板截图画廊。
只要把你的 4 张截图按下面的文件名放进这个 images/ 文件夹，
HTML 打开后就会自动显示真实截图；没有放图片之前会显示带标签的占位框。

请使用以下文件名（区分大小写，建议用 PNG）：

  1. dm-1-overview.png        ——  全览（KPI 卡片 + 预警事项）
  2. dm-2-drafts.png          ——  底稿进度
  3. dm-3-confirmations.png   ——  函证进度
  4. dm-4-os.png              ——  OS 进度

提示：
- 文件名必须完全一致；如需改名，请同时修改 HTML 中对应的 <img src="..."> 。
- 图片建议宽度 1600px 以上，横向截图（约 2:1）效果最佳。
- 放好图片后无需改动其它内容，刷新页面即可看到。
